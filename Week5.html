Lesweek 5

Eerste stap richting coderen!

Opdracht: neem een screenshot van je ontwerp,
en zet het in ChatGPT. Zet daarna je HTML en CSS code in Visual Studio Code.

Prompt in ChatGPT:

  • Maak hiervan een HTML-bestand zodat het visueel hetzelfde is.
  • Dat ziet er al goed uit! Kun je dit nu met HTML en CSS doen?

Het ontwerp dat ChatGPT had gemaakt was bijna helemaal hetzelfde als het origineel,
het enige kleine detail dat miste was het blauwe lijntje onder elke bestemming.
Telkens als ik vroeg om dat te veranderen maakte hij een heel ander ontwerp, dus heb ik het voorlopig zo gelaten.

Goedgekeurd door de leerkracht!
